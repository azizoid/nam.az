## Nam.az

### www.nam.az is a prayer time tracker for Azerbaijan with the major cities

## Tech Stack
- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Jest**
- **GitHub**
- **GitHub Actions**
- **MongoDB**
- **Google Maps API**
